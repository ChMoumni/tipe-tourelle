# TIPE – Tourelle de visée motorisée

Projet de conception d’une tourelle motorisée pilotée par joystick, réalisé dans le cadre de mon TIPE (2025-2026).

## 📁 Contenu du dépôt

- `code/` : code Arduino C++ et simulation Wokwi (schéma + programme).
- `cad/` : modèles 3D des pièces mécaniques conçues sous FreeCAD.
- `images/` : photos du montage réel et captures d’écran de la simulation.

## 🛠️ Technologies utilisées

- Arduino / C++ embarqué
- Wokwi (simulation)
- FreeCAD (modélisation 3D)
- Électronique : soudure, câblage, servomoteurs
